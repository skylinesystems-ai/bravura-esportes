# BRAVURA ESPORTES // ARENA DIGITAL

Frontend em React com Vite, Tailwind CSS, Framer Motion e Lucide React para um catálogo futurista com foco em conversão pelo WhatsApp.

## Rodar localmente

```bash
npm install
npm run dev
```

Depois abra a URL exibida pelo Vite. Neste projeto, o script usa `http://localhost:5189`.
